# Toy Store KPI Report

## Business Problem Outline
Maven Toys is a toy store chain with multiple locations in Mexico. They provided data containing transactional records from January 2022 to September 2023 along with information about products and store locations. They want a simple, interactive report that the leadership team can use to monitor key business metrics and high-level trends.

## Skills
Power BI, Measures, DAX, Power Query, Data Modeling
